## üöÄ Cara Menjalankan (Local Development)
Untuk menjalankan project ini di komputer lokal Anda:

1.  **Clone Repository**
    ```bash
    git clone https://github.com/smartonesda/smartonesda.github.io.git
    cd smartonesda.github.io
    ```

2.  **Jalankan Website**
    Karena ini adalah *static site*, Anda bisa langsung membuka file `index.html` di browser.
    
    *Disarankan menggunakan **Live Server** (ekstensi VS Code) agar aset termuat sempurna.*
